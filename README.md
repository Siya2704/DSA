- Search a node
- Height of bst
- Print tree

- Count leaf nodes
- List leaf nodes
- Level order traversal
- Display level of nodes
- Return max value
- Convert bst to.max heap
- Mirror of bst
- Check if tree is bst
- Get width of a level
- Get width of tree
- Compare two tree
- Check if bst is avl tree

- Build heap from array
